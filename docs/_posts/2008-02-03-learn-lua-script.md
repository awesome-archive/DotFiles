## 起因

大学期间，和朋友一起玩过一段时间《魔兽世界》，这款游戏的插件都是使用 lua 来写的。
当时折腾插件时经常莫名其妙报错。于是，就想着要不自己也学习下 lua，然后也可以自己写一些插件。


## 开始学习

最初的时候，其实并没有太多的在线教程，只能硬着头皮看官方的手册。好在 lua 算是比较简单的语言。

电脑上是 Ubuntu 7.04 LTS 版本，装上 lua 和 Vim 这两个工具后，就开始学着写一些简单的脚本了。
